# 需求变更：全部页面改为手机横屏布局 + 标准德州扑克桌面

## 一、全局横屏适配
1. 所有页面（首页、等待页、对局页）改为**横屏布局**，CSS 用 `landscape` 媒体查询 + `100dvh/100vw` 横向铺满。
2. 加一个 **竖屏检测遮罩**：当检测到 `orientation: portrait`（宽<高）时，全屏覆盖一个遮罩，显示一个旋转手机图标 + 文字"请横置手机游玩"，阻止所有交互。横屏时遮罩消失。
3. PC 端遮罩逻辑保留（"请使用手机浏览器访问"），但 PC 在横屏宽屏下也显示旋转提示遮罩。
4. 移除之前竖屏专用的 `max-width:480px` 等限制，改为横屏铺满。

## 二、对局页标准德州桌面布局（app/pages/game/[id].vue）
这是核心改动，做一个**标准德州扑克椭圆桌面**：

### 桌面区域
- 屏幕中央一个**椭圆桌面**（CSS `border-radius: 50%` 或椭圆比例），深绿色/深蓝色桌面纹理，金色边框。
- 桌面中央上方：**5 张公共牌**横向排列（未发的牌显示为牌背占位）。
- 桌面中央：**底池总筹码**（大字显示，带筹码图标）。
- 桌面中央下方：**当前轮次提示**（Preflop/Flop/Turn/River/Showdown）。

### 玩家座位（最多 6 人，环绕桌面）
标准 6 人桌座位分布（横屏视角）：
```
        座位2(上方)
  座位1          座位3
(左上)            (右上)

  座位6          座位4
(左下)            (右下)
        座位5(下方/自己)
```
- **座位5（下方居中）永远是"我"（当前玩家本人）**，手牌完全可见、操作按钮在正下方。
- 其余 5 个座位按顺时针排列环绕桌面。
- 每个座位显示：
  - 玩家昵称
  - 剩余筹码
  - 当前下注（如果有，显示在座位与桌面之间，带筹码视觉）
  - 状态标识：在线/离线、弃牌(Folded)、全下(All-In)、思考中(倒计时)
  - **庄家按钮(D)**、**小盲(SB)**、**大盲(BB)** 角标
- 非本人座位手牌默认背面朝上；摊牌阶段后端返回亮牌时翻开。
- 当前行动玩家的座位**高亮**（发光边框/脉冲动画），并显示**60秒倒计时进度环**（围绕玩家头像或座位边框）。

### 底部操作区（横屏底部居中，本人座位下方）
- 轮到自己时显示：弃牌、跟注（显示金额）、加注（滑块或数字输入 + 确认）、全下。
- 不轮到自己时操作区隐藏或灰显。
- 操作按钮用大号触摸友好尺寸（min-height 44px）。

### 对局日志（右上角或右侧侧边栏，可折叠）
- 竖向滚动日志列表，显示操作记录。
- 默认收起，点击展开。不遮挡桌面。

### 断线重连按钮
- 放在左上角小图标，不显眼但可点。

## 三、首页横屏布局（app/pages/index.vue）
- 左右分栏或居中卡片布局，适配横屏宽屏。
- 左侧/上方：昵称输入 + 创建房间。
- 右侧/下方：可加入房间列表（横向排列或网格）。
- 整体居中，不过度拉伸。

## 四、等待页横屏布局（app/pages/room/[id].vue）
- 玩家列表横向排列或网格（2x3），适配横屏。
- 房主按钮在底部或右侧。

## 五、CSS/样式要求
- 用 CSS 变量定义桌面色、牌面色、筹码色等主题色。
- 牌面用 CSS 画（不引第三方库）：白底圆角，花色用 Unicode 符号（♠♥♦♣），红心方块红色、黑桃梅花黑色。
- 牌背用深色渐变 + 花纹。
- 筹码用 CSS 圆形渐变模拟，不同面值不同颜色。
- 倒计时进度环用 CSS `conic-gradient` 或 SVG circle stroke-dashoffset。
- 座位高亮用 `box-shadow` 脉冲动画。
- 整体不引入任何 UI 组件库，纯 CSS。

## 六、技术约束
- 不改后端 API 和游戏逻辑，只改前端页面布局。
- 保留 3 秒轮询、token 重连等现有逻辑。
- `npm run build` 必须通过。
- commit: `feat: landscape layout with standard poker table`
- push origin main。

## 七、验收
1. 手机横屏打开对局页 → 看到椭圆桌面、6 个座位环绕、公共牌在中央、操作按钮在底部。
2. 竖屏时显示"请横置手机"遮罩。
3. PC 端显示"请使用手机浏览器访问"遮罩。
4. 当前行动玩家座位高亮 + 倒计时环。
5. 摊牌阶段他人手牌翻开。
6. 首页、等待页横屏布局正常。
