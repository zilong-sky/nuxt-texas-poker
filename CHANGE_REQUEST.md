# 需求变更：参照参考图优化对局页布局

## 参考图分析结果（2340x1080 横屏）
通过对参考图的像素级分析，得出以下布局特征：

### 整体
- 深黑背景，中央一个**大椭圆绿色桌面**占据屏幕约 80% 面积
- 桌面金色边框，内部深绿色felt

### 桌面中央
- **公共牌**：5张白色牌横向排列在桌面上半部（约 y=25-30% 处）
- **底池**：金色文字显示在公共牌下方（约 y=40% 处）
- 底池旁边有蓝/金色筹码视觉

### 玩家座位（环绕桌面）
- **顶部**：2-3个玩家座位横向排列（y=5-15%）
- **左侧**：1个座位（y=40%, x=5%）
- **右侧**：1个座位（y=40%, x=92%）
- **底部**：本人座位在桌面下方居中，但不占太大空间
- 每个座位：圆形头像 + 昵称 + 筹码，紧凑显示
- 下注筹码显示在座位与桌面之间

### 底部操作栏（关键改动）
- **4个彩色大按钮横向排列**：
  - **弃牌**：红色背景 (background: linear-gradient red)
  - **跟注/过牌**：金色/黄色背景，显示金额
  - **加注**：蓝色背景 + 金额输入
  - **全下**：白色/浅色背景，显示筹码数
- 按钮高度约 50-60px，圆角，有立体感
- 本人手牌在操作按钮上方居中显示

### 座位布局（6人桌，横屏）
```
     座位1        座位2        座位3
   (左上)        (中上)        (右上)

  座位6                        座位4
  (左中)                        (右中)

         座位5(本人+操作栏)
          (底部居中)
```

## 具体改动要求

### 1. 桌面放大
- `.table` 宽度从 70% 改为 **85%**，高度从 72% 改为 **80%**
- 桌面 `border` 加粗到 10-12px，金色更亮
- 内部 felt 绿色用径向渐变加深边缘

### 2. 座位位置调整（app/pages/game/[id].vue CSS）
重新定义 6 个座位的绝对位置：
- `.seat-1`：top: 3%, left: 15%（左上）
- `.seat-2`：top: 1%, left: 50%, transform: translateX(-50%)（中上）
- `.seat-3`：top: 3%, right: 15%（右上）
- `.seat-4`：top: 42%, right: 2%（右中）
- `.seat-6`：top: 42%, left: 2%（左中）
- 座位5（本人）不在此区域，移到底部操作区

### 3. 座位卡片精简
- 每个座位改为：头像(36px) + 右侧紧凑信息（昵称一行 + 筹码一行）
- 标签(D/SB/BB)用小角标贴在头像角落
- 下注筹码显示在座位朝桌面中心方向

### 4. 底部操作区重做
- 本人手牌：2张大牌居中显示在操作按钮上方
- 本人信息（昵称+筹码）在手牌左侧或右侧
- **4个彩色按钮**：
  ```css
  .btn-fold { background: linear-gradient(135deg, #e53935, #b71c1c); color: white; }
  .btn-call { background: linear-gradient(135deg, #ffb300, #ff8f00); color: #1a1a1a; }
  .btn-raise { background: linear-gradient(135deg, #42a5f5, #1565c0); color: white; }
  .btn-allin { background: linear-gradient(135deg, #f5f5f5, #e0e0e0); color: #1a1a1a; }
  ```
- 每个按钮 min-width: 80px, min-height: 48px, border-radius: 10px
- 加注按钮内嵌一个数字输入框（宽度 80px）
- 按钮间距 10px，整体居中

### 5. 倒计时环
- 当前行动玩家的座位头像外圈显示 conic-gradient 倒计时环
- 颜色用金色，从满到空

### 6. 日志面板
- 默认隐藏，右上角放一个小"日志"图标按钮
- 点击后从右侧滑出半透明面板

### 7. 不改的部分
- 后端 API、游戏逻辑、轮询机制全部不动
- 只改 `app/pages/game/[id].vue` 的 template 和 style
- `app/components/PokerCard.vue` 不改
- `app/assets/main.css` 里如果需要新增 CSS 变量可以加

## 验收
1. `npm run build` 通过
2. 横屏看：大椭圆桌面占满，6座位环绕，底部4个彩色操作按钮
3. commit: `feat: optimize game layout per reference image`
4. push origin main
