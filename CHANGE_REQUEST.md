# 德州扑克 UI 大调整（严格对标参考图，只改前端视觉）

**范围**：只改 `app/pages/game/[id].vue`（必要时可加 CSS）。**不动** `server/`、KV schema、房间/开局/超时/筹码/盲注等业务逻辑；不动 `app/pages/index.vue` / `app/pages/room/[id].vue`。

**页面方向**：游戏页强制横屏（已有 rotate hack 保留）。所有布局按横屏视口设计。

## 一、椭圆牌桌基础样式
1. 修正当前椭圆被竖屏比例拉伸变形的问题：`.table-outer/.table-inner` 使用固定宽高比的横向椭圆（宽:高 ≈ 16:9 或 2:1），`border-radius: 50%/50%`，居中放置。桌面深蓝色底(#0e2a52 类)，桌沿一圈黑色软包边框(粗一点、内阴影)。桌面中央叠加半透明浅色 "HOLD'EM" 字样水印。
2. 椭圆桌**顶部正中间**固定放卡通女荷官（可用现有 avatar/emoji 占位），位置不动。荷官**正下方居中**显示本局底池总筹码（金黄色数值，M 缩写）。
3. 椭圆桌**左上角**：庄位标识 "D" 圆形筹码片（白底蓝字或红字）+ 一张红色牌背卡片作为装饰，还原参考图。（庄家指示 D 若庄家不在席位 3/4 附近，此左上角 D 是装饰性固定位；如需精确跟随庄家，也可在庄家席位手牌旁另放一枚 D 筹码。**优先固定左上角装饰位**，简单可靠。）

## 二、6 人环形席位（顺时针，我方永久锁在下方左侧）
1. 席位 1：桌面**下方左侧**（我方固定，不可变）
2. 席位 2：桌面**右下角**
3. 席位 3：桌面**右上角**
4. 席位 4：桌面**顶部右侧**（荷官右手边）
5. 席位 5：桌面**顶部左侧**（荷官左手边）
6. 席位 6：桌面**左下角**（我方左侧）

### 席位统一 UI
- 圆形头像框，尺寸一致，不变形不被裁切；
- 头像右上角紫色礼物礼盒小图标（🎁 圆形紫底）；
- 头像下方横向胶囊显示筹码数量（金黄色，M 缩写），紧贴头像下缘、可略微叠压头像底部，视觉一体；
- **不显示玩家昵称文字**；
- 除我方外的 5 席，手牌默认背面遮挡，牌下小字「暂时无法查看」，只有摊牌阶段亮牌；
- 玩家下注/全下时，座位侧边展示红色牌背卡片标记；
- AI 机器人和真人玩家外观规范一致。

## 三、公共牌区
- 椭圆桌**水平居中**位置横向平铺公共牌区：翻牌 3 张 → 转牌 4 张 → 河牌 5 张依次显示；
- 扑克牌带纸张光影/阴影，视觉厚实；
- 位置在荷官下方、底池筹码下方；确保不被我方手牌/庄家 D 遮挡。

## 四、我方手牌
- 我方两张手牌固定在**我方头像正前方**（即席位 1 头像上方靠桌内一点的位置）——不是屏幕底部、不是桌下空白区、不是桌面中央；
- 手牌下方紧接：牌型名称（高牌/一对/两对/三条/…）；
- 牌型下方：彩色胜率进度条（渐变红→黄→绿）；
- 我方轮到行动时，手牌旁边显示 60s 倒计时环。
- **关键**：整个"我方手牌+牌型+胜率条"块紧贴席位 1 头像上方（往桌子中央方向），像牌摆在我面前的桌面上一样，不要脱离席位 1 独立浮到别处。

## 五、底部操作按钮（横向 4 个）
- 页面**最底端横向排布**（当前如为竖向必须改回横向）；
- 4 个深蓝色圆角按钮，白色中文字，从左到右：**【弃牌】【过牌/弃牌】【过牌】【跟任意注】**；
- 按钮宽度均分或接近，手指点击友好（min-height 44px）；
- 根据对局状态自动 disable 置灰（沿用现有 `foldDisabled` / `checkDisabled` 等逻辑）。
